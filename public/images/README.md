# Screenshots pour Landing Page

## Instructions

Pour compléter l'intégration de la landing page, vous devez ajouter les vrais screenshots suivants :

### 1. dashboard-screenshot.png
- Accédez à `/dashboard` en production
- Prenez un screenshot haute résolution
- Nommez le fichier `dashboard-screenshot.png`
- Placez-le dans ce dossier (`/public/images/`)

### 2. pipeline-screenshot.png
- Accédez à `/pipeline` en production
- Prenez un screenshot haute résolution
- Nommez le fichier `pipeline-screenshot.png`
- Placez-le dans ce dossier (`/public/images/`)

## Résolution recommandée
- Format : PNG
- Largeur : au moins 1200px
- Qualité : haute résolution pour un rendu net dans le browser mockup

## Note temporaire
En attendant les vrais screenshots, la landing page utilisera des images de fallback ou pourra générer des erreurs de chargement d'image. Ceci est normal et sera résolu une fois les screenshots ajoutés.
