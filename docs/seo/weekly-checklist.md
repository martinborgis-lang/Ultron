# Checklist SEO Hebdomadaire — Ultron CRM

> **Objectif:** Routine de monitoring pour atteindre 9.5+/10 SEO Score
> **Durée:** 2-3h/semaine réparties sur 5 jours
> **Période:** Lundi → Vendredi (30-45min/jour)

---

## 🔧 LUNDI - Technical SEO & Health Check
**⏰ Durée estimée:** 45 minutes

### 🔍 Google Search Console (15min)
- [ ] **Erreurs de couverture** - Vérifier nouvelles erreurs/warnings
  - 404s détectées (pages supprimées/déplacées)
  - Erreurs de crawl (serveur, redirections)
  - Pages bloquées par robots.txt
- [ ] **Performance** - Analyser 7 derniers jours vs semaine précédente
  - Clics, impressions, CTR moyen
  - Requêtes en baisse/hausse
  - Pages les plus impactées
- [ ] **Amélioration pages** - Core Web Vitals alerts
- [ ] **Sitemaps** - Statut indexation sitemap.xml
- [ ] **Mobile Usability** - Nouveaux problèmes détectés

### 🔍 Bing Webmaster Tools (10min)
- [ ] **Erreurs de crawl** similaires à GSC
- [ ] **Backlinks** nouveaux détectés
- [ ] **Mots-clés** performance vs Google

### ⚡ Core Web Vitals - PageSpeed Test (15min)
**Pages prioritaires à tester:**
- [ ] Homepage: https://ultron-murex.vercel.app/
- [ ] Landing features: /features
- [ ] Blog principal: /blog
- [ ] Page contact: /contact

**Métriques à noter:**
- [ ] **LCP Mobile/Desktop** (objectif: <2.5s mobile, <1.8s desktop)
- [ ] **INP Mobile/Desktop** (objectif: <200ms mobile, <100ms desktop)
- [ ] **CLS Mobile/Desktop** (objectif: <0.1)
- [ ] **PageSpeed Score** (objectif: 85+ mobile, 95+ desktop)

### 🕸️ Liens Cassés (5min)
- [ ] Test rapide liens internes pages principales
- [ ] Vérification redirections (301/302)
- [ ] Check images/ressources manquantes

**📝 Actions Lundi:**
- Noter dans KPI tracker issues critiques
- Planifier fixes techniques prioritaires
- Alert dev si problèmes bloquants

### 📝 Content Ideas Collection (5min)
- [ ] Sujets tendances secteur CGP via Google Trends
- [ ] Questions clients récurrentes (support/ventes)
- [ ] Angles nouveaux sur mots-clés existants

**📝 Actions Mardi:**

## 🚨 Alertes & Escalations

### Alertes Critiques (Action Immédiate)
- **🔴 Chute trafic organic** > 30% en 24h
- **🔴 Site down/crawl errors** > 50% pages
- **🔴 Core Web Vitals** échec total mobile
- **🔴 Pénalité Google** suspectée (rankings -70%+)

### Alertes Importantes (Action 48h)
- **🟡 Rankings tier 1** chute > 10 positions
- **🟡 Backlinks** perte > 5 liens DA>30
- **🟡 Indexation** problèmes > 20% pages
- **🟡 Performance** dégradation > 20%
